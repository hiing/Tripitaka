# 起世因本经(10卷)

此经宋藏题为起世经，闍那崛多共达摩笈多译，进之为前经编入澄函，而丹藏题为起世因本经，达摩笈多译，却之为后经。今捡《开元录》，丹藏为正，故题加「因本」二字，译除闍那崛多，却之为后经，为取函焉。
